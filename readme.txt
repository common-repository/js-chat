== Installation ==

This section describes how to install the plugin and get it working.

   1. Upload wp-click-ckeck.php to the /wp-content/plugins/js-chat/ directory.
   2. Activate the plugin through the ‘Plugins’ menu in WordPress
   3. Create your Chat room on https://js-chat.appspot.com/
   4. Copy & Paste the endpoint url to your chat room into the wordpress backend.
